# Crypto Dashboard

A **Python dashboard** to visualize the prices of popular cryptocurrencies like **Bitcoin** and **Ethereum** over the past few days. This project fetches real-time data from the [CoinGecko API](https://www.coingecko.com/en/api) and plots interactive charts using **Matplotlib**.

---

## Features

- Fetches real-time cryptocurrency prices from CoinGecko
- Plots price trends over the last 7 days
- Supports multiple cryptocurrencies
- Saves the chart as a **PNG file**
- Clean and readable Python code
- Fully compatible with Windows, Linux, and macOS
